🧠 Level 2: Vibe Coding Basics

Learn the golden rules that actually matter!

🔥 The 4 Golden Rules (Print This Out!)

🎯

Google > Memory

You don't need to memorize. You need to know WHAT to search.

📋

Copy → Understand → Modify

Don't reinvent. Find working code, understand it, adapt it.

🐛

Debugging = Real Skill

Writing code is 20%. Fixing it is 80%.

🎈

Small Wins > Big Ideas

Make ONE button work. Then add features.

🚀 The 6-Step Vibe Coding Workflow

1

💡 IDEA

What problem are you solving? Keep it simple!

↓
2

🏗️ SKELETON

Create basic files: index.html, style.css, script.js

↓
3

⚡ CORE FEATURE

Make ONE main thing work (even if ugly)

↓
4

📦 EXPAND

Add 2-3 more features, one at a time

↓
5

🐛 DEBUG

Fix errors, test everything, break it on purpose

↓
6

✨ POLISH

Make it look good, add CSS, prepare presentation

🧠 Mindset Shift (Why People Fail)

❌ Wrong Thinking

  • ❌ "I need to understand everything"
  • ❌ "I must write perfect code"
  • ❌ "I should know all commands"
  • ❌ "My code must be original"

✅ Correct Thinking

  • ✅ "I need to make it WORK"
  • ✅ "Ugly working > pretty broken"
  • ✅ "Google when I forget"
  • ✅ "Learn from others' code"
🎯 KEY TAKEAWAY: Your goal is to make something WORK, not to write perfect code. Perfection comes with practice!
