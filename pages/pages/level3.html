⚡ Level 3: Terminal Mastery

Command the terminal like a pro hacker!

🔥 TRUTH BOMB: If you don't understand terminal → you'll feel LOST in every project!

📁 Navigation Commands

These 4 commands will get you 90% of the way:

pwd

Shows current folder

Example: /Users/you/Desktop
ls

Lists all files & folders

See what's in current folder
cd folder-name

Go inside a folder

cd Desktop → enters Desktop
cd ..

Go back one level

Like pressing "back" button

📁 File Creation Commands

# Create a folder mkdir my-project # Create a file (Mac/Linux) touch index.html # Create a file (Windows) type nul > index.html # Create multiple files at once touch index.html style.css script.js

🎯 Real Example Workflow

Follow this exact sequence to create your project:

$ cd Desktop # Go to Desktop
$ mkdir hackathon-project # Create folder
$ cd hackathon-project # Enter folder
$ touch index.html style.css script.js # Create files
$ ls # See files you created
index.html style.css script.js

🔥 Pro Tips

  • Tab key is magic: Start typing folder name, press Tab to auto-complete
  • Up arrow: Recalls previous command (no need to retype!)
  • Clear screen: Type clear (Mac/Linux) or cls (Windows)
  • Stop a running program: Press Ctrl + C
