🌐 Level 6: Frontend vs Backend

Understand how the web really works!

🎭 The Simple Analogy

🏪

Frontend = Store Front

What customers SEE and INTERACT with

  • ✓ Display of products
  • ✓ Shopping cart
  • ✓ Pretty design
  • ✓ Buttons & forms
↔️
🏭

Backend = Warehouse

Behind-the-scenes LOGIC and STORAGE

  • ✓ Inventory database
  • ✓ Payment processing
  • ✓ User accounts
  • ✓ Business logic

→ API Call →

🧪 Real Example: Login Flow

1️⃣ Frontend:

User clicks "Login" button

fetch('/api/login', { method: 'POST', body: { email, password } })
↓
2️⃣ Backend:

Receives request, checks database

if (password matches) { return success_token } else { return error }
↓
3️⃣ Frontend:

Receives response, shows dashboard

if (success) { redirect('/dashboard') } else { show('Wrong password!') }
💡 Key Insight: Frontend = What users see and click. Backend = The brain that does calculations and stores data. They talk via APIs!
